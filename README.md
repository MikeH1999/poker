# River Club

一个适合朋友私下娱乐的实时积分制德州扑克网页游戏。无需注册，创建房间后复制链接即可邀请好友入座。

## 本地运行

```bash
npm install
npm run dev
```

打开 <http://localhost:3000>。分享链接给同一网络中好友时，需要把 `localhost` 换成这台电脑可被访问的地址；部署到公网后，邀请链接会自动使用线上域名。

## 已实现

- 2–8 人私人房间与 6 位房间码
- 可配置起始积分、大小盲和人数上限
- 服务端洗牌、发牌和操作校验
- 翻牌前、翻牌、转牌、河牌与摊牌
- 弃牌、过牌、跟注、加注和全下
- 牌型判断、平分底池与边池结算
- 同一赢家的边池奖金合并展示，未被跟注的下注自动退回
- All-in 后公共牌按时间间隔逐张发出，并带翻牌动画
- 每位玩家实时查看自己的“底牌 + 公共牌”组合牌型
- 断线重连、房主移交、牌局动态和文字聊天
- 房主暂停/恢复牌局、调整盲注、修改玩家积分和剔除玩家
- 房主可开启结算展示 5 秒后自动开始下一手
- 玩家持续离座与“我回来了”状态（返桌后从下一手发牌）
- 默认 1000 积分，玩家首次入座前可自行填写入座积分
- 邀请链接采用两步入座：先查看 8 个座位并选择空位，再填写昵称和积分
- 已占座服务端锁定，断线重连保留原座位，剔除后座位自动释放
- 流式响应式桌面/H5 界面，支持小屏、大屏和手机横竖屏

## 公平机制

服务端只向每位玩家发送自己的底牌。其他玩家（包括房主）只能在摊牌后看到仍在局中的牌。项目不包含隐藏看牌入口或房主作弊后门。

## 自动化检查

先启动服务，再在另一个终端运行：

```bash
npm run test:smoke
```

脚本会创建两个实时客户端并完整打完一手，检查底牌隔离和积分守恒。
